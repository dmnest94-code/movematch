# MoveMatch v0.3 — First Customer Release

Static HTML/CSS/JavaScript website deployed through Netlify.

## Structure

- `index.html` — landing page and Build My Move wizard
- `css/styles.css` — visual system and responsive layout
- `js/app.js` — wizard, validation and form submission
- `js/recommendation-engine.js` — recommendation rules
- `privacy.html` — starter privacy policy
- `success.html` — successful submission page
- `netlify.toml` — deploy and security configuration

## Netlify

Build command: leave blank  
Publish directory: `.`

Form name: `move-request`
